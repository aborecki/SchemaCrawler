** Connect to jdbcodbc_nz

      --database=<database>
                          Database name
      --host=<host>       Host name
                          Optional, defaults to localhost
      --port=<port>       Port number
                          Optional, defaults to 50000
      --server=<server>   --server=jdbcodbc_nz
                          Loads SchemaCrawler plug-in for jdbcodbc_nz
